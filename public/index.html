HOME:

HTTP METHOD DESCRIPTION
' /(home page) ' The Static Page provides all the routes

USERS:

HTTP METHOD DESCRIPTION
'/users' GET Returns an array of all users
'/users/:id' GET Returns single user's information
'users/register' POST Registers new user
'users/update/:id' PATCH Updates user's information
'users/login' POST Logs in existing user and sends back jwt
'users/delete/:id' DELETE Removes user's information

POSTS:

HTTP METHOD DESCRIPTION
'/posts' GET Returns an array of all posts
'/posts/:id' GET Returns single post
'posts/add' POST Adds new post
'posts/update/:id' PATCH Updates post's information
'posts/delete/:id' DELETE Removes post

COMMENTS:

HTTP METHOD DESCRIPTION
'/comments' GET Returns an array of all comments
'/comments/:id' GET Returns single comment
'comments/add' POST Adds new comment
'comments/update/:id' PATCH Updates comment
'comments/delete/:id' DELETE Removes comment
