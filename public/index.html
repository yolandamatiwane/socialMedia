HOME:

HTTP METHOD DESCRIPTION
' /(home page) ' The Static Page provides all the routes

USERS:

HTTP METHOD DESCRIPTION
'/users' GET Returns an array of all users
'/users/user/:id' GET Returnsan object with single user's information
'users/register' POST Registers/adds a new user
'users/user/:id' PATCH Updates user's information
'users/login' POST Logs in existing user and sends back jwt
'users/user/:id' DELETE Removes user's information

POSTS:

HTTP METHOD DESCRIPTION
'/posts' GET Returns an array of all posts
'/posts/:id' GET Returns single post
'/posts' POST Adds new post
'/posts/post/:id' PATCH Updates post's information
'/posts/post/:id' DELETE Removes post

COMMENTS:

HTTP METHOD DESCRIPTION
'/post/:postId/comments' GET Returns an array of all comments as per post id
'/post/:postId/comment/:cID' GET Returns single comment as per post id
'post/comment' POST Adds new comment
'/post/:postID/comment/:cID' PATCH Updates comment by authorized user
'/post/:postID/comment/:cID' DELETE Removes comment by authorized user
'/post/:postID/comment' DELETE Removes comments from a post by authorized user
