USERS:

HTTP METHOD DESCRIPTION
'/users' GET Returns an array of all users
'/users/:id' GET Returns single user's information
'users/register' POST Registers new user
'users/update/:id' PATCH Updates user's information
'users/login' POST Logs in existing user and sends back jwt
'users/delete/:id' DELETE Removes user's information
